Certamen 02

El propósito de este trabajo consiste en la exploración de la visualización representativa de bases de datos. Para este certamen se utilizo la base de datos anual entregada para crear dos visualizaciones con simetría en torno al centro, lógica artística de una Mandala, utilizando los datos de: Temperatura Máxima, Temperatura Media, Temperatura Mínima, Punto de Rocío, Rocío Promedio y finalmente rocío Mínimo. Este código utiliza la rotación y la traslación del punto de origen para crear un centro vacío y entregar un mayor enfoque a los datos visualizados en un ángulo de 360 grados en torno a este.

Es importante notar que los valores originales fueron ampliados en cada visualización (en múltiplos de 15.6 y 10.5 respectivamente) con el único fin de presentar la información de una manera más visible para el usuario.

La interactividad entre el usuario y la visualización es simple y consiste en dos comandos:

Tecla Numérica ¨1¨: Activación de Visualización 1
Tecla Numérica ¨2¨: Activación de Visualización 2
